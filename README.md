# SpaceWebRadio

Small web radio application  
GUI: pygame  
radio: mpd/mpc   


Started on GitHub: 13.05.17


cd /home/pi    
sudo git clone https://github.com/vektorious/SpaceWebRadio.git  
cd SpaceWebRadio  
sudo chmod +x install.sh  
sudo ./install.sh  

after the installation finishes do a reboot

finally execute
>mpc load stations

to load the stations
